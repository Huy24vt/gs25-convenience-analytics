# GS25 Convenience Store Analytics

## Project Overview
End-to-end retail analytics simulation of a GS25-style convenience store using synthetic transaction data.

## Business Questions
- When are peak demand hours by store location type?
- Which categories drive revenue vs profit?
- Are promotions effective?
- How should staffing and merchandising be optimized?

## Key Findings
- Office stores peak at lunch; residential stores peak in evening.
- Beverage & RTE drive revenue.
- Snack & Dessert drive margin.
- Promotions increase basket size but reduce margin.

## Business Recommendations
- Optimize staffing by area_type and hour.
- Focus promotions on high-margin add-on products.
- Redesign discount-based promo to protect gross profit.
- Track gross profit KPI, not only revenue.
